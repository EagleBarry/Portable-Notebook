### **A simple Chrome extension for saving and organizing text across multiple notebook style tabs. **

## Features:

Create multiple tabs for separate notes
Automatically save content per tab
Lightweight and fast
Persistent storage using Chrome Sync

## Quick Tips:

Renaming Tabs - Double-click on any tab name to rename it.

Select text then right-clcik in order to send it to one of your tabs.

## Right-Click Actions

Right-click next to the tabs in order to:

-Send the data from the Active tab to another tab

-Clear the current tab content

-Close a tab

## How it works

Each tab is an independent notebook page. Your notes are automatically saved and synced using Chrome storage, this way your data persists between sessions.
